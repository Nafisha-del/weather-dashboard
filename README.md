# Weather Dashboard App

A React weather dashboard that allows users to search real-time weather by city.

## Features
- City search
- Current weather
- Temperature
- Humidity
- Wind speed
- Error handling
- Responsive UI

## Tech Stack
- React
- JavaScript
- OpenWeather API
- Vite
- GitHub
- Vercel
